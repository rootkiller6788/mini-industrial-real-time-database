# Course Alignment — Time-Series Compression Algorithms

## 9-School Curriculum Mapping

| School | Course | Coverage |
|--------|--------|----------|
| **MIT** | 6.302 Feedback Systems / 2.171 Digital Control | Sampling theory (Nyquist), DFT/FFT, signal processing for control systems |
| **Stanford** | ENGR205 Process Control / EE392 Industrial AI | Facebook Gorilla (Pelkonen et al., VLDB 2015), time-series databases, industrial compression |
| **Berkeley** | ME233 Advanced Control / EE C128 Mechatronics | Digital signal processing, FFT algorithms, embedded compression |
| **CMU** | 24-677 Advanced Control Systems / 18-771 Linear Systems | Shannon information theory, entropy coding, Huffman trees, system identification from compressed data |
| **Georgia Tech** | ECE 6550 Nonlinear Control / AE 6530 Optimal Estimation | Transform-domain processing, frequency-domain compression, Kalman filtering for adaptive deadband |
| **Purdue** | ME 575 Industrial Control / ECE 602 Lumped Systems | OSIsoft PI architecture, Bristol swinging door patent, industrial historian systems |
| **RWTH Aachen** | Industrial Control Systems / PLC-SCADA Engineering | OPC UA data access, quality flags, historian compression in Siemens/Bosch ecosystems |
| **Tsinghua** | 过程控制工程 / 工业互联网 (Process Control / Industrial IoT) | Time-series databases, signal processing for process monitoring, compression for industrial big data |
